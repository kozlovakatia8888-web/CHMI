🔤 Типографика в UI: типы шрифтов и параметры читаемости

Serif, sans-serif, моноширинные. Размер, длина строки, интерлиньяж, контраст — как не испортить интерфейс.

Типы шрифтов

  • Serif (с засечками) — Times New Roman, Georgia. Для длинных статей, заголовков. На экране при малых размерах читаемость может страдать.
  • Sans-serif (без засечек) — Inter, Roboto, Helvetica. Стандарт UI: кнопки, поля, навигация, мобильные интерфейсы.
  • Monospace — JetBrains Mono, Fira Code. Для кода, терминалов, таблиц. Не использовать для абзацев.

Критические параметры читаемости

ПараметрРекомендация
Размер (font size)Основной текст ≥16px (десктоп), 16px (мобильные). Заголовки 24–32px.
Длина строки (line length)50–75 символов (десктоп), 35–50 (мобильные). Контроль через max-width.
Интерлиньяж (line height)1.4–1.6 от размера шрифта. Пример: line-height: 1.5.
Контраст (contrast)WCAG AA: обычный текст ≥4.5:1. Плохо: #AAAAAA на белом (2.3:1).
🔧 Дополнительно: избегать длинных фраз в uppercase (читаются медленнее), трекинг по умолчанию 0, выравнивание по левому краю (не по ширине).
